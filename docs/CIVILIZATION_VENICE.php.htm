威尼斯

历史

“最尊贵的威尼斯共和国”建立在亚得里亚海里一个咸水潟湖的岛群上。在中世纪和启蒙时代，它成为了欧洲最大的港口，同时也是欧洲和东方经济和文化交流的纽带。据考证，最早的威尼斯人是躲避日耳曼和匈人入侵的罗马难民。在公元726年，威尼斯居民揭竿而起，宣布脱离拜占庭帝国的统治，独立为城邦共和国，随后选举了威尼斯历史上117任中的第一任督治。从9世纪到12世纪，威尼斯逐渐发展为经济和军事强权，支配了包括黎凡特（地中海东部）地区、远东地区、摩纳哥和西班牙在内的地中海各条贸易线路。威尼斯的商人和探险家们向四面八方探索，足迹覆盖了亚洲和非洲，给城市带来了更加丰厚的财富，同时也使威尼斯成为欧洲的文化中心之一。在地中海沿岸，威尼斯的海员们甚至敢于挑战拜占庭和奥斯曼的霸权。最终，由于包括新大陆的发现以及欧洲各王朝间征战之内的各种因素，威尼斯的财富、影响力和力量都开始消退。1797年，拿破仑率军占领了这座城市，从此它就再没有作为独立国家出现过。尽管如此，在灭亡之后，威尼斯在人们的集体想象之中依然占据着不可取代的位置。大运河、大理石教堂、壁画装饰的宫殿以及众多杰出的艺术作品营造的光景，依然倒映在蓝天下威尼斯潟湖的水波里，使威尼斯成为现代社会中最传奇的名字之一。

气候与地形

威尼斯坐落于亚得里亚海东北端的新月形威尼斯潟湖内，由117个小岛组成。威尼斯潟湖大约有32里长，从耶所罗湿地一直延续到南端的基奥贾镇。潟湖的水由一排有三个缺口的沙丘遮蔽，因此城市水域能够有涨潮退潮，也允许进出船只。潟湖分为两个部分，活水和死水区。后者由波河和皮亚韦河以及几十股小溪流留下的沉积物构成的咸水河与沼泽组成，被利多岛从海洋隔开，是潟湖的主要部分。威尼斯气候温和，这是由其北面的阿尔卑斯山和南面的阿德里亚海的天气形态决定的。夏季气温平和，平均约为80华氏度（译注：约27摄氏度），而冬天的平均气温约为30华氏度（译注：零度左右），并时常伴有大雾或薄雾。全年平均降水约34英尺。威尼斯城已经建立了数个世纪，因此岛上极少有野生的动植物生存。

共和国建立之前

尽管没有关于威尼斯第一批移民的记载，罗马史学家指出，5世纪时，在伦巴第和匈奴人的入侵下，罗马难民从帕多瓦、阿奎莱亚和崔维索之类的城市撤离到这个潟湖的盆地里。在那里，这些罗马人和当时往来作业的渔民与盐矿工一同生活，在几个岛上建立了村庄。一般认为城市本身建立以421年3月城市中第一座基督教教堂——里阿尔托岛圣贾科莫教堂的建立为标志。城市属于拉文纳主教区的一部分，由君士坦丁堡的拜占庭皇帝指派一名总督管理。公元726年，主教区的人民揭竿而起反抗教皇格里高利二世，此时教皇正着手于将东方正教传播到西方。在起义的混乱中，威尼斯潟湖的居民们宣布独立，并选举了第一位领导人。居民们选举出的领导人，乌尔苏斯，是威尼斯117位督治中的第一位，来自城市的最上层家族。

名门望族对督治权力和地位的长期争夺，并没有放缓这个海上强权在贸易事业上的蓬勃发展。在舰队的保护下，威尼斯人在地中海的几个大港口都建立了贸易站，财富从各地不断涌入城市，远至斯拉夫的基督教王国和西班牙与北非的穆斯林之地。收入的提升使得城市的统治阶级扩张，督治的权力受到限制，使人们开始拥有民族认同感，从而使国家更加稳定。不论多任教皇、查理曼大帝和伦巴第的丕平如何尝试削减甚至试图终结威尼斯的经济和军事发展，城市还是无可避免地走向了繁盛。在775年，教皇和威尼斯督治达成一致，在威尼斯的奥利弗拉岛上建立了教皇管区。有两件事情可以用于证明，在这个时期里威尼斯的声望和影响，其发展速度快于世界上的其他国家：一是832年，圣马可教堂建成，用于存放的基督教福音传播者圣马可（威尼斯的保护圣徒）的遗物，而这遗物是由威尼斯商人从亚历山大里亚偷来的。另一件事情是几个岛屿上开始广泛建造防御工事，以防卫整个潟湖和威尼斯城本身。

中世纪的威尼斯

在10世纪早期，共和国开始着手一项巨大的造船工程，以位于威尼斯城市中心，面积高达110亩的船坞——威尼斯兵工厂为核心地点。在这个时代，威尼斯有36000名海员，驾驶着3300艘战舰，是整个地中海规模最大的海军。到1200年，威尼斯已经占领了一些城市，并沿着阿德里亚海在达尔马提亚和伊斯特里亚建造了戒备森严的前哨站。一连串持扩张论者的督治甚至指挥占领了威尼斯北面大陆的一些地区，以保证威尼斯向阿尔卑斯方向的贸易畅通，同时也保证城市的小麦供应。为了掌控盐的贸易，凭借地中海最强大的海军力量，威尼斯从日渐衰落的拜占庭帝国处夺得了包括塞浦路斯和克里特岛在内的爱琴海大部分岛屿。以中世纪欧洲的标准来说，威尼斯在其领土上的管制相对开明，周边城镇如贝加莫、布雷西亚、维罗纳和拉古萨等的居民也会热心支持威尼斯的战争和探险贸易事业。

在第四次十字军东征后，威尼斯成为了强权国家。年事已高且双目失明，但充满智慧的督治恩里科·丹多洛决定“拿起十字架”并让威尼斯加入十字军，并亲历了1204年君士坦丁堡被占领并将其洗劫一空的经过。在他去世的前一年，丹多洛在随后的瓜分拜占庭的谈判中扮演了重要角色，使威尼斯获得了新的领地，从拜占庭和十字军国家两头都获得了贸易特权。

在中世纪早期，威尼斯共和国的政府形式和罗马共和国在很多方面有类似之处，体现在它的大议会上，这是城市里最古老和最富裕的家族的集合。议会任命所有官员，并选举一个包含200到300人的参议院。为了处理国家的重要事务，大议会中还选举出一个包括督治在内的十人议会。一般市民，理论上来说，有权力支持或拒绝对主要职位（包括督治本身）任命人选的提议。

文艺复兴

君士坦丁堡于1453年陷落于奥斯曼土耳其人之手，这将威尼斯推上了欧洲抵抗奥斯曼土耳其入侵的最前线。1499年，土耳其攻占了威尼斯城市摩里亚，一方面使他们能直接接触亚得里亚海沿岸，另一方面也使他们和威尼斯永无和解的可能。在接下来的一个世纪里，当土耳其人在扼杀威尼斯和东方和非洲的贸易时，威尼斯人却陷入和土耳其争夺霸权的泥潭中。这样的冲突直到1571年10月的勒班陀战役后才得以结束，在那场战斗中，威尼斯率领天主教舰队决定性地打败了土耳其舰队。尽管这场胜利保证了土耳其人不会再向西侵略，但它同时也标志着威尼斯长久衰落的开始。

尽管如此，威尼斯的财产和富饶依然声名远扬，威尼斯的贵族也是14到16世纪间意大利文艺复兴的主要资助人。作为世界性的城市，威尼斯群星荟萃，聚集了一大批如提香之类的艺术家，乔凡尼贝利尼和丁托列托等生长于威尼斯的天才，加布里埃利之类的作曲家，和隆格纳（最著名的巴洛克风格设计师之一）之类的建筑家。在此同时，一大批先进知识分子，例如在著名的帕多瓦大学任教的伽利略，也在自由的威尼斯领地得到了庇护。印刷机这一崭新的发明也在这个时期在欧洲传播，在1428年，威尼斯已经是世界的“印刷之都”。城中的首席印刷师阿尔杜斯·马努蒂乌斯，制造了第一本一般人士买得起的平装书。威尼斯的政治正在衰落，但它文化上的声望却毫无损耗。就像1500年米兰的一个旅行者写下的，威尼斯是“我见过的最伟大的城市……根本不可能用语言描述这座独一无二城市的美丽、壮观和富有。”

衰落

威尼斯的衰落有着各个方面的原因。黑死病在1348年、1575-1577年和1630年三次在威尼斯爆发，而最后的一次杀死了约50000名居民，约是当时城市人口的三分之一。其他国家，例如葡萄牙、英国、荷兰、斯堪的纳维亚在造船和海军方面的进步也使得威尼斯的战船和贸易船只过时。在1498年，葡萄牙探险家瓦斯科·达·伽玛成功绕过了好望角，在欧洲和东方之间开辟了一条海上通路。之后，葡萄牙、英国和荷兰的商人们迅速取代了威尼斯人在欧洲经济中的位置。与此同时，西班牙人正从新世界里带来威尼斯人无法匹敌的财富。西班牙、法国和神圣罗马帝国为了意大利半岛的霸权征战之时，威尼斯的政治影响不断减弱，军队的繁重事务正在抽空共和国的积蓄。

到了公元1508年，教皇、匈牙利人、萨瓦人和费拉雷斯人为了对抗威尼斯在地中海的霸权而联合组成了康布雷同盟。尽管由于同盟的内部分裂，威尼斯没有被灭亡，但它大部分的大陆领土已经丢失。在勒班陀的胜利尽管于一段时间内挽救了威尼斯的国际声望，但导致了土耳其人回过头来完成他们对东地中海的占领。在四年的战役之后，1669年威尼斯丢掉了意大利向东的最后一块领土——克里特岛。尽管随后为了重建国家发起了一系列成功的战役，顶峰时甚至收复了摩里亚，在人力和资源上威尼斯都付出了惨痛的代价。在1718年，之前费尽千幸万苦夺回的领土全部被还给了奥斯曼人，以交换贸易优惠。

在接下来的几十年里，威尼斯的发展完全停滞了，它活在昔日的荣光之下。接下来一连串无能的督治使得威尼斯在政治、经济和外交上都被全面孤立。它注定会被被法国大革命的风暴消灭。拿破仑，军事上冷酷无情的实用主义者，声称这个城邦在他1797年对抗奥地利的战争中是撤退时的一大阻碍。标志奥地利战败的里奥本和约使威尼斯失去了对抗法国勒索的有效盟友。1797年三月，威尼斯的最后一任督治卢多维科·马宁在拿破仑的压力下退位。这一年的晚些时候，根据卡波夫米奥条约，威尼斯被割让给奥地利。

共和国之后

在1848年，威尼斯城紧接着横扫整个欧洲的革命浪潮，又建立了临时共和国。这个共和国存在的时间不长，在次年被奥地利吞并。在1866年，随着奥地利被普鲁士人打败，威尼斯转归给五年前已经统一的意大利管辖。从此以后，威尼斯的命运就和意大利联系在一起，经历了两次世界大战、法西斯主义、冷战结束和欧盟兴起。虽然如此，威尼斯依然是我们这个星球上存活时间最长的共和国。

逸闻

“红发神父”，小提琴名家安东尼奥·卢瓦尔第一生作品众多，但他的大部分作品写给了圣女慈济所的巴洛克女合唱团。后者是威尼斯本地的一所修道院，收留孤儿，同时也是一所音乐学校。

威尼斯方言里的“督治”(doge)实际上是拉丁语“领袖”(dux)的一个变音。在英语中对应的词语是“公爵”(duke)，尽管两个头衔的能力和义务从来就不能相比。最大的差别在于前者是选举产生的，而后者则是世袭的。

在中世纪，威尼斯经济的核心是“伙会”，一种由商人建立起的合股公司，用于资助单一的长途贸易。这样的组织形式允许那些大胆的投资者们——包括艺术家、工匠、寡妇、平民和学生——有机会从成功的贸易中获得高额回报。
