历史信息:

铸币厂是一个文明生产其流通货币的机构。货币在过去通常是硬币形式的金属，到了现代社会世界上大部分的货币都是纸质的。铸币厂对一个文明的经济健康运作极其重要。如果一个国家的货币可以被轻易地伪造，将会导致人们对货币供应的信心下降，进而引起危险的通货膨胀（货币的实际价值降低）甚至更糟的情况。现代钱币采用全息技术和变色油墨印刷并附带安全丝来防止伪造，毫无疑问还将引入新的技术，因为伪造者正在破解现有的防伪技术。
