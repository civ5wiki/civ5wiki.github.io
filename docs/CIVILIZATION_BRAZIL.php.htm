巴西

历史

巴西最伟大的小说家保罗·柯艾略这样描述他的人民的历史，“他们在搜寻命中注定的宝藏，没人想碌碌无为苟活一生。”在所有的美洲前殖民地中，巴西是独特的。因为其不仅是通过相对和平的方式获得独立，更是没有像大英帝国和西班牙的殖民地那样分崩离析成很多国家。从公元1500年佩德罗·卡布拉尔到达美洲宣布巴西为其君主所有开始，直到1815年巴西都是葡萄牙的殖民地，那年葡萄牙皇室从被占领的葡萄牙逃离到巴西，巴西成为一个王国。到了1822年则实现了完全独立，时年一个君主立宪制和国会制政府的巴西帝国建立了。到了1899年，伴随着其第二任也是最后一任皇帝佩德罗二世的下台，巴西成为了一个共和国。虽然在接下来的一个世纪里，共和国经历了独裁和军阀的磨难，在1980年代，真正的民主得以回归。在权力过渡的谈判完成后，第一届民选政府上台了。接下来在一系列杰出的继任领袖领导下，巴西政治和经济趋于稳定，成为国际舞台上重要和有影响力的一员，最终找到了“[他的]命运的财富”。

气候与地形

巴西是世界第五大国家，占整个南美洲陆地面积的一半，，是有“多样性生态环境”的17个国家之一，有着多种多样的动植物、栖息生物、自然资源和地形。亚马逊河流盆地的大部分都在巴西，这是世界上最大的河流系统和最大的原始森林。所以，巴西有着一系列的热带和亚热带景观，包括湿地、热带草原、丛林高原以及低山地形。沿着巴西的7500英里海岸线分布着众多群岛。国土的南半部多为山地占据，一系列低山在那里升起如曼蒂凯拉山和埃斯皮尼亚苏山，最高点在4000英尺。由于位于赤道线上，整个国家呈现出热带气候，可进一步分为五种亚类型：赤道、热带、高地热带、温带、亚热带。巴西的平均温度为华氏77度，植被覆盖从北边的雨林，到中间的热带草原，再到南方的松叶针林，一路变化。有着如此多样性的地形和气候，巴西的生物多样性在世界上首屈一指，美洲虎、豹猫、貘、食蚁兽、树懒、犰狳、鹿、水虎鱼、凯门鳄、鹦鹉、猴等成数千物种在陆地和水域繁衍生存。

葡萄牙的殖民

巴西原住民（葡萄牙人称之为indios）的起源无从可考，最早的人类化石可追溯到10000年前的米纳斯吉拉斯高地。当葡萄牙探险家到达海岸时，有大约2000个土著部落生存在巴西。他们过着半游牧生活，以狩猎、捕鱼、季节性农业、部落战争甚至食人为生。1500年4月，佩德罗·卡布拉尔宣布这块土地归葡萄牙所有。当时他率领船队从好望角附近一直向西，经历了长途航行后，最终在南美大陆登陆。公元1494年的托尔德西里亚斯条约沿着46度经线将新世界瓜分为西班牙（西部）和葡萄牙（东部）所有。由于这次发现正好位于协议规定的葡萄牙区域内，同时受到关于当地富饶的报告的刺激，1532年，第一个葡萄牙定居点建立了。

巴西红木的发现激起了皇室的兴趣，这是一种厚重、橘红色的硬木，在染料和乐器、家具制作领域深受好评。公元1534年，若昂三世国王鼓励私人殖民冒险。到了1549年，他任命了一位总督将军，巴西从此正式成为葡萄牙殖民地。在与法国人的一系列战争中，葡萄牙人缓慢得向南北两个方向扩展自己控制的领土，分别在1567年和1615年占领了里约热内卢和圣路易斯。到了1680年，葡萄牙人宣布拉普拉塔河附近的土地归为己有，这是他们获得的最南端的领土;与此同时，还攻陷了英国人和荷兰人在亚马逊雨林的要塞；土著部落则要么被同化，要么被贩为奴隶，要么被彻底根除消灭。

到17世纪末时，巴西已经成为葡萄牙分散在世界各地的殖民地中最大的也是最重要的一个。除了巴西红木之外，糖、染料和香料也是主要出口产品。当国际市场开始出现对奴隶商品的需求时，葡萄牙人开始从非洲进口奴隶。最终，葡萄牙成为奴隶贸易的主要国家之一，而巴西的奴隶数量则达到了数十万计。同时，勘探者们在巴西的丛林和山脉上寻找金矿矿脉，并最终在米纳斯吉拉斯发现了可观的储量。接下来的淘金热开采出巨额的黄金，以至于在1763年殖民地首府从南部的萨尔瓦多迁移到了里约热内卢，以更好得管理这新的财富。1750年的马德里条约，1761年的帕尔多条约和1777年的伊尔德丰索条约划定了巴西的边界，与此同时殖民地也进行了一系列改革，确保了巴西继续作为一个平静繁荣的殖民地给葡萄牙带来收益。

独立

1808年3月，拿破仑的军队占领了葡萄牙，葡萄牙王室和大臣们抵达里约热内卢避难。由于玛丽亚一世女王因精神疾病无力执政，他的儿子若昂王子成为摄政王代为执政，在里约热内卢重建了首都并继续帝国的统治。在他的统治期间，一个主权国家的首都应有的政府部门都落实到位，他还创建了一座皇家图书馆，一所军事学院，一个皇家铸币厂，一个印刷机构，以及一些医学院和法学院。1815年，若昂宣布巴西作为一个王国，在帝国内与葡萄牙地位等同。在法国战败后，他更愿意留在巴西，直到被召回葡萄牙处理激进分子的叛乱。1821年4月，若昂任命他的儿子佩德罗为摄政王。佩德罗的大臣们有许多是在巴西出生，力劝他独立。1822年9月，这位年轻的摄政王发表了一份巴西独立宣言，之后不到3个月，他被加冕为佩德罗一世皇帝。1825年，葡萄牙政府承认了巴西的主权，接下来不到一年时间，大部分欧洲国家也相继承认其主权。

帝国

佩德罗一世和他的大臣们想要确保巴西免受纷争和革命的荼毒，而这些正在其南美邻国上演。基于这个目标，一部全新的宪法由此诞生，他是主要构造者。这是一部在当时相当开明和先进的宪法。但是佩德罗越来越多得参与到了葡萄牙的事务中，1831年他宣布退位由其5岁的儿子继位，这样他便能返回葡萄牙为其女儿争取葡萄牙的王冠。为了解决因佩德罗一世突然离去而留下的政治动荡和纷争，他的儿子被官方宣布为14岁，并在一年内加冕为佩德罗二世皇帝。这位新皇帝接下来50年的统治开明且进步，巴西在各个领域，从政治到经济到工业、社会、文化，都进入了黄金时代。在佩德罗二世治下，巴西赢得了三场战争的胜利，提高了国际声望，变得现代化，同时还改革了法律和财政体系，增强了农业多样性，废除了奴隶制。但是废除奴隶制损害到了地主阶层的利益，而且随着佩德罗二世年岁越来越高，他慢慢失去了与新兴城市中产阶级和自由派学生运动的联系，而这些正是他的理念和政策培育出来的产物。虽然仍然受到人民的尊敬和爱戴，1889年11月的一场兵不血刃的军事政变废黜了他，取而代之的是一个共和国。作为一个永远的爱国者，即便是在逃离巴西流亡欧洲时，他仍然表达出自己“看到伟大而繁荣的巴西的热切希望”。

独裁

领导这次政变的德奥多罗·达·丰塞卡成为了临时总统，而政府中的军人势力也与日俱增。无论是建立共和国，还是政教分离，亦或是发布新宪法，他都主要依赖军方和日益繁荣的咖啡种植园主们的支持。虽然作为临时总统，外界却宣称其无法有效履行应有职责。正当其试图分化新的立法机构并推行法令时，却被迫在公众的强烈抗议下辞职。他的副总统，同样是一位将军，接替了总统职务。在平息了一些军方和保皇派的叛乱之后，这位总统的独裁权力与日俱增。

1894年，在一片和平氛围中，佩绍托将军不情愿得向第一位民选总统普鲁登特·德·莫拉伊斯移交了总统权力。他曾是富产咖啡的圣保罗州州长，被视为是众多“咖啡总统”中的第一位。这些总统主要来自圣保罗和米纳斯吉拉斯的富有政客和地主，他们积极推动经济改革、基础设施现代化、维持和平以及采用一种近乎孤立主义的政策引导巴西渡过动乱的国际时期。但是，真正的民主是少之又少，只有拥有土地的少数人才有资格投票，不公正的选举随处可见，区域性的政治寡头只要支持总统就能获得实质上的豁免权。

两场运动最终终结了这“咖啡总统”时期。首先，咖啡价格在1930年代世界范围的大萧条中一落千丈。其次，一场由低级军官（陆军中尉）发起的运动影响力与日俱增。这些陆军中尉支持民粹主义，拥护的不是民主而是改革和进步。他们发自内心地认为只有军队才能把这个国家推进到现代。基于这样的出发点，这些年轻的军官们计划驱逐民选政客，扩大联邦政府的权力，推进军队现代化，通过一个强有力的中央政府来根除地方主义。

大萧条和这种动荡不安使得瓦加斯、这位落选的总统候选人在陆军中尉的支持下夺取了权力。本来他只是在经济危机期间暂时掌握权力，然而，他关闭了国会，废止了宪法，巴西各州的州长都被替换为支持他的人，以军官居多。1935年的一场共产主义政变失败了，1938年的一场法西斯政变也失败了，瓦加斯的政权进化成了全面的独裁，以其残忍和对新闻出版的审查而闻名。巴西在1942年以积极作战的姿态加入了盟军，在意大利驻有大量地面部队，在大西洋则有大规模海军。尽管是一个独裁政权，瓦加斯坚称基于在第二次世界大战中的表现，巴西是联合国的创始国之一。

但是战后的经济繁荣和国际环境却使瓦加斯的统治动荡不安。1945年他被另一场军事政变赶下台，15年前中止了民主的利益集团这次则是“恢复”了民主。讽刺的是，1950年瓦加斯再次当选为总统，但是却在1954年的一场政治危机中于卡台提宫自杀了。瓦加斯死后，紧随而来的是一系列短命的政府，成就和腐败并存。1964年，另一场军事政变颠覆了民选政府。到了1968年，第五制度法案授予了军方势力全面的独裁权力。虽然手段很严酷，但是军政府却不似南美大陆上其他国家的政府那样残忍。更重要的是，军政府还促进了资本主义、现代化和国际协议。因此，即使是在那些镇压的年月里，军政府依然在中低阶层深受欢迎。

现代巴西

埃内斯托·盖泽尔将军在1974年接过了总统权力，并立即启动了“缓慢、渐进而安全”得过渡到民主政府的政策。在接下来的几年里，他慢慢终止了对政治犯的拷问、对新闻媒体的审查，最终撤消了第五制度法案，军政府也不复存在。他的继任者延续这个道路，若泽·萨尔内在1985年的第一次自由选举中当选为总统。从那时起，凭借其稳定的政府和繁荣的经济，巴西在地区事务中开始扮演领导角色，成为一系列组织的创始成员，如拉丁语联盟，美洲国家组织、南美共同市场以及南美国家联盟。新千年伊始，巴西已经成为国际舞台上平等而有影响的一员。

逸闻

巴西利亚是世界上为数不多的特意修建的首都之一，始建于公元1956年，源于时任总统库比契克·德奥利韦拉根据巴西的第一部共和国宪法中的某一条款要求迁都，这部宪法可上溯到1891年。该条款指出国家的首都应位于国土中心。经过41个月的建设，政府于1960年4月迁移到了新首都巴西利亚。

作为举国欢庆的节日，巴西狂欢节贡献了全国年度旅游贸易额的70%。为期40天的庆祝活动，充斥着游行、化妆舞会、街道节日、舞蹈以及不时举行的宗教仪式，桑巴和斧乐更进一步加强了节日的气氛。

从1500至1700年间，由葡萄牙和巴西探险家带领的先锋旗手们，在他们搜寻奴隶、黄金和稀有植物的路上，探索了大部分亚马逊雨林并绘制了地图。同时，在雨林深处建立了基地，使得巴西的西部疆域稳固地向外扩张。
