荷兰

历史

（塞爱维小提示：荷兰是个约定俗成的称谓，其正式名称是“尼德兰”，意为“低地之国”，“荷兰”仅是当初尼德兰的七省之一。中文史籍通常在其独立前称“尼德兰”，独立后称“荷兰”以作区分。）
荷兰或称“尼德兰”，从欧洲北方海岸线上处在多个强大帝国的势力范围之内的小小行省崛起为世界最重要的贸易中心之一。从罗马的征服开始，其后的法兰克的崛起，以及最后的神圣罗马帝国，荷兰频繁的成为扩张的欧洲王国的目标。然而，他们留下的印迹都无法超越西班牙哈布斯堡家族——腓力国王对荷兰新教徒的迫害直接导致了八十年战争的爆发。将荷兰的人民推上了反抗西班牙宗主追求独立的长达一个世纪的斗争。自从建立了在政治上独立自主的国家之后，荷兰人民一直不知疲倦地努力发展自己的民族特色。而在这个新生的共和制国家逐步成为世界经济的主导力量的历史进程中，也诞生了不计其数的文化和科技伟人。

地理和气候

荷兰王国处在欧洲北部的海岸，同时也管理着加勒比海中的阿鲁巴，库拉索和圣马丁岛。荷兰在欧洲的省份海拔非常的低，许多的土地是通过扩展大堤从海里获得的。尽管荷兰的土地相对较小，大约只有美国的新泽西面积的两倍，那里却居住着超过一千六百万的人民。由于可用的土地非常的小，荷兰有着悠久的城市规划和建设的历史，许多荷兰人也居住在包括首都阿姆斯特丹在内的高密度的城市中心里。

史前时代

在罗马人于公元前一世纪到来之前，现今的荷兰地区居住着许多的日耳曼部落。这些部落来自于周边地区，尤其是斯堪的纳维亚地区。他们同时也是发源于德国和奥地利地区的塞尔特人的早期后裔。在这些部落中，居住在莱茵河上的一个岛屿上的巴达维人，是最强大和令人恐惧的。尽管巴达维人并不被认为在贸易和工艺上特别成功，但仅仅是他们在战斗中的声望就足以另他们在周边部落里取得统治地位。

罗马占领时期

荷兰最南部的区域，即罗马人的下日耳曼尼亚省，在公元1世纪时被罗马人以武力占领。根据凯撒大帝的《高卢战记》记载，巴达维部落曾经作为雇佣军协助罗马人在欧洲作战。然而，巴达维部落最终由于罗马人越来越紧密的控制发生了叛乱，导致了对他们的镇压。罗马的军队控制了这个区域长达接近400年没有受到任何挑战，直到帝国整体的崩溃。

弗里斯兰和法兰克

尽管这个时期的文字记录非常罕见，在罗马帝国衰弱后，荷兰境内的帝国领土迅速被法兰克人和弗里斯兰人取得。尽管这些部落在这个地区居住了数个世纪，只有在一度强大的罗马帝国的缺席下，他们扩张的机会才大大增加。法兰克帝国最终取得了几乎整个荷兰和德国，仅留下处于北部海岸的弗里斯兰。由于居住的如此接近，法兰克和弗里斯兰最终成为了苦涩的对手，通过一系列的战争来解决领土的争端。在734年，法兰克在伯尔南之战胜利宣告了弗里斯兰王国的终结并将这些土地吸收入法兰克帝国。法兰克随后对意大利的征服和在查理曼大帝时期的扩张奠定了一些未来出现的王国的基础，包括法国和德国。在随后的几个世纪里，荷兰的土地处于他们的统治之下。

帝国变迁

在查理曼大帝于814年去世后，法兰克帝国分成了三个部分，中法兰克，东法兰克和西法兰克王国。荷兰的大部分成为了洛泰尔一世治下的中法兰克帝国的一部分。这个曾经庞大的帝国的分裂同时也导致了许多王国在德国地区（后来成为了神圣罗马帝国）和法国地区诞生，他们都曾经为了在荷兰地区的影响和土地而战。当这个地区的许多部分最终在公元15世纪被神圣罗马帝国控制时，勃艮第公爵这个法国王室的旁枝，开始觊觎这片土地。

同时也在这个时期，许多可怕的洪水第一次袭击了荷兰，造成了千百人的死亡和基础设施的大量毁坏。在1287年，灾难性的圣卢西亚洪水袭击了荷兰，造成了超过50000人的死亡并把一个小湖泊直接变成了北海的一部分，即须德海。须德海直到20世纪才被从北海分离，恢复了它曾经淡水湖的情况。

权力的转移

在14和15世纪，荷兰以及邻近的比利时几乎完全被勃艮第公爵控制。整个十五世纪，这些法国公爵持续的通过联姻继承，精心计算的购买以及直接吞并持续不断的从低地国家获取了越来越多的土地。

勃艮第在荷兰的土地最终通过勃艮第女公爵玛丽和奥地利的马克西米利安一世的联姻被哈布斯堡家族获得。尽管马克西米利安一世对勃艮第的继承权受到法国国王的反对，这些土地最终还是被玛丽的儿子，腓力一世继承，确保了权力向哈布斯堡的转移。

1521年，哈布斯堡家族分裂为西班牙哈布斯堡和奥地利哈布斯堡。西班牙哈布斯堡继续控制着荷兰的土地，而奥地利哈布斯堡则通过数任神圣罗马帝国皇帝控制了德国和周边区域。

新教革命

在欧洲新教改革期间，当罗马天主教的传统权威受到质疑时，新教运动在荷兰境内留下了深刻的印记。荷兰的天主教统治者对荷兰境内新教信仰的传播感到非常不安，起先是查理五世，随后是更重要的他的儿子，西班牙国王腓力二世。腓力国王对荷兰新教徒越来越残酷的镇压最终在其后的几年里造成了荷兰人民不满的爆发。

对西班牙的起义

在对腓力国王政策的不满在荷兰传播时，荷兰革命最显著的标志，奥兰治的威廉成为了宗教容忍的大胆支持者。威廉自己是作为新教徒长大的，但是作为他贵族教育的一部分，他改宗了天主教。尽管他的信仰不同，他坚定地相信所有人都有实践自己信仰的权力。威廉是荷兰的贵族，是西班牙忠诚的下属，他最初对反抗国王的命令感到犹豫。然而，当腓力丝毫不试图安抚荷兰人民的不安时，威廉被迫作出抉择。到1567年时，小规模的叛乱已经开始行动，威廉由于选择背弃腓力国王并资助起义被西班牙方面宣布为罪犯。

当威廉领导他的军队在正面取得了对西班牙的数次胜利后，联省共和国于1581年在北部被解放的土地上建立。然而，这些最初的成功只是标志着被称为八十年战争的长期冲突的开始，在这段时期内，西班牙国王依然维持着对荷兰南部区域的控制。奥兰治的威廉在1584年被一个忠于腓力国王的天主教徒刺杀，而战争则持续到1648年，最终荷兰共和国从西班牙的控制下完全独立了出来。

贸易霸主

在八十年战争和荷兰共和国的建立后，荷兰迅速成为了欧洲统治性的贸易中心。在这个世界上第一个跨国公司，荷属东印度公司的带领下，荷兰的商人在全球建立了新的贸易路线和殖民地。这些被认为是荷兰的黄金时期的日子，也见证了荷兰的科学进步，军事创新以及境内蒸蒸日上的艺术和音乐文化。

海洋争霸

当英国越来越关注荷兰与英国、法国和西班牙殖民地的贸易对英国贸易努力和利润的损害时，英国议会于1651年通过了一系列的法案来限制荷兰船队的扩展。这些法案被称为“航海法案”，用来限制荷兰对英国港口的访问。英国这些控制海洋的努力直接导致了17世纪被称为英荷战争的一系列的冲突。在随后的100年里直到18世纪中期，由于无数的海战导致荷兰舰队数量和力量上的衰弱，荷兰在政治和世界贸易中的力量遭到了极大的削弱。

王国的建立

在拿破仑战争时期，荷兰在1795年被法国吞并，建立了法兰西共和国的卫星国，短命的巴维亚共和国。在1814年枫丹白露协定的签署正式宣告拿破仑征服的终结后，荷兰人民又一次通过成立荷兰联合王国来宣告他们的独立。随后在1830年，比利时，这个直到那时还一直被认为是荷兰一部分的地区，也脱离荷兰成为了一个独立的国家。

世界大战

从建立主权国家之初，荷兰就一直试图在任何可能的情况下维持一个严格中立的政策。作为结果，荷兰在第一次世界大战中保持了中立，没有在战争中出兵但是依然和双方都维持了经济关系。

在第二次世界大战中，荷兰在冲突爆发时又一次宣告中立。然而这次，德国人不再允许有价值的荷兰土地保持不结盟。在1940年，纳粹入侵荷兰，他们起初遭到了荷兰军队的一些抵抗，但是荷兰最终还是被第三帝国蹂躏并占领。在德国占领期间，荷兰政府逃离了这个国家并组成流亡政府，而这个国家则被纳粹军事占领。尽管当德国的占领变得越来越令人无法忍受时荷兰的抵抗运动也在不停的发展，纳粹依然在荷兰杀死了成千的荷兰犹太人和来自德国的犹太难民。其中包括安妮·弗兰克，她的日记成为了犹太大屠杀的广为传阅的编年史。

在1944年，为了夺取莱茵河以及默兹河上从荷兰通往德国的大桥，盟军发起了历史上最大的空投作战——市场花园行动。盟军在这个区域空投了数千的空降兵，然而行动却不是很成功。盟军指挥官们把失败归因于阻止盟军推进的坏天气和通讯问题。作为这次失败的结果，荷兰直到1945年春天德国最终投降之前始终在纳粹政权的手中。

现代荷兰

现代的荷兰王国是一个君主立宪国家，拥有一个民选的议会来协助执政女王贝娅特丽克丝。作为欧盟的创始国之一，荷兰王国长期为欧盟给这个地区带来和平和经济稳定的努力作出贡献。荷兰的第三大城市海牙，既是荷兰政府的所在地也是联合国国际法庭的所在。

文化标志

在历史上，荷兰创造了许多世界上最杰出的艺术家、科学家、研究人员和学者。他们中的许多人都参与奠定了其所在领域现代的发展的基础。著名画家伦勃朗·范·莱因和文森特·梵·高都是荷兰人，如今他们的作品都在艺术界最杰出和有价值的作品之中。科学家中则有着发现了土星环的克里斯蒂安·惠更斯和“微生物学之父”安东尼·范·列文虎克，他们都为未来的科学探索作出了极大的贡献。

逸闻

荷兰最大的出口商品之一，郁金香，在整个国家都有生产。在十七世纪荷兰的经济力量处于巅峰时，一次被称为“郁金香狂热”的事件，激发了郁金香贸易中广泛的投机和价格操纵，使稀有的郁金香品种的价格在最终回落前到达了一个天文数字。

全球都在消费的喜力啤酒，最早于1873年在阿姆斯特丹创建。如今和它标志性的红色和绿色商标一起已经成为一个被广泛认同的荷兰的标志产品。

荷兰这个名字据说来源于荷兰语的词汇“holtland”，意思是长满树的土地。
