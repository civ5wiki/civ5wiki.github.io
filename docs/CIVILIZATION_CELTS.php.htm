凯尔特

历史

欧洲的凯尔特人，这个以今天世界上数百万人的祖先而被纪念的部落，曾经是历史上最伟大的古代文化族群之一。据说从欧洲中部早期铁器时代文明发展起来的凯尔特人，逐渐成长为一个强大而又多元化的由遍布大陆的数百个单独部落组成的族群。在公元前最后一个千年的大部分时间里处于鼎盛时期的凯尔特人最终卷入了与无情的，最终将镇压他们的但是在征服欧洲的过程中在很多方面继承了他们文化的罗马人的一系列的可怕冲突之中。

地理与气候

尽管凯尔特人最终散布到了包括法国（高卢），不列颠群岛，西班牙（伊比利亚半岛）的大部分欧洲，已知最早的凯尔特人居住在今天的德国和奥地利地区。气候和地理特征的巨大变化使他们的领土很难用几个具体特点来描述，尽管这的确证明了凯尔特人有能力适应这样巨大变化的坚强本质。

凯尔特人的起源

尽管凯尔特人祖先的起源依然被今天历史学家们争论着，他们被认为发展出了哈尔施塔特的瓮棺文化和在青铜时代晚期和铁器时代早期继承了它的欧洲的拉特尼文化。这些早期的凯尔特人迁徙范围十分广阔，并把它们的语言和宗教与和他们的相遇的原住民相融合。在大约公元前1300年，第一个原始凯尔特族群，瓮棺文化，主要被发现于今天德国的最南端。生活于青铜时代晚期的瓮棺文化人在原始酋长的统治下，最终扩张到了法国。在瓮棺文化的人口增长和领土扩张过程中，松散从属的各部落之间经常发生冲突，而且对于处于持续扩张和整合阶段的部落来说打败自己的偏远族人的现象并非少见。

之后发展出来的以奥地利哈尔施塔特村附近的考古发现命名的哈尔施塔特文化，继续着原始凯尔特人在欧洲的扩散。在公元前最后一千年期间，早期凯尔特人持续着穿越中欧的移民，并发展成了所谓的在铁器时代后期扩张到欧洲大部的拉特尼文化。从拉特尼开始，最早的有显著特征的凯尔特文化族群在大陆的一些部落联盟间形成了。

不列颠的凯尔特人

岛屿凯尔特人，是指那些主要生活在不列颠列岛，也许是曾经散布整个欧洲的凯尔特人族群中当今最大的遗族。在公元前7世纪到3世纪，被共同的语言和宗教所联系起来的铁器时代的凯尔特人广泛的定居于不列颠、苏格兰以及爱尔兰。尽管已经和当地土著融合，凯尔特移民继续散布，最终成为了这个地区的居民。

公元1世纪凯尔特人在艾西尼的布狄卡女王的旗帜下联合起来并爆发出可怕的力量。在忍受了罗马人的无端残暴后，布狄卡领导着一支不列颠群岛部落联军对抗着罗马的军团，当时最强大的军事力量。自从公元前387年被凯尔特酋长布雷努斯击败以来，罗马人首次遭遇如此的由布狄卡的成千上万人组成的凯尔特军队。尽管最终罗马人成功的镇压了叛乱（为此双方损失总计超过十万人），正如之前的历史所展示的，罗马人又一次的低估了凯尔特人的力量并为此付出了高昂的代价。

凯尔特伊比利亚人

伊比利亚半岛（现在的西班牙，葡萄牙和安道尔）的凯尔特人是从高卢迁徙过去的。在公元前6世纪和5世纪早期离开他们的家乡后，他们很快融入了当地业已存在的伊比利亚文化。凯尔特伊比利亚人，是第一次布匿战争中迦太基对抗罗马时的联盟，并以此而出名，也因这个决定使他们最终在著名的迦太基将领汉尼拔战败后被罗马人征服。

凯尔特伊比利亚人以建造丘堡（Castro），一种在伊比利亚半岛上建造的石山堡垒而闻名。尽管多种形式的丘堡在凯尔特人的居住的不列颠和中欧地区发现，在今天西班牙发现的则保存得特别完好。努曼提亚，一个由凯尔特伊比利亚人建造于公元2世纪的丘堡，已经成为了现代西班牙国家性的纪念碑。

高卢人

尽管被罗马人征服之前的高卢历史因缺乏准确的记录而有所局限，高卢人可能依然是最为人们所熟悉的凯尔特人。主要居住在今天法国地区的高卢人被尤里乌斯凯撒在他的关于高卢战争的历史著作高卢战记中被频繁的描述。由许多单独的小部落组成的高卢人在他们的小城镇之间发展出了全面的贸易路线，由此建立起来的文化独立地繁荣了几个世纪。尽管如此他们丰富的文化被罗马人最终的征服完全毁灭。

尽管在欧洲的所有凯尔特族群中都可以发现德鲁伊，他们在高卢人中特别享有盛誉。作为一种在凯尔特人中享有极高威望的祭祀阶级，德鲁伊在凯尔特人社会中扮演了多种重要角色：包括法官和学者，以及他们作为宗教人物的传统地位。不幸的是，德鲁伊的睿智只流传于口头，导致宗教仪式的神秘遗产的失传，其中许多至今无法解释。

罗马的征服

公元前390年，高卢酋长布雷努斯攻击了罗马的盟友伊特鲁里亚并最终攻到了罗马，并包围了城市准备毁灭它。在收到一笔慷慨的赎金后布雷努斯放弃了他的征服，他和凯尔特人们在他们的胜利中发了一笔横财。但这只不过是罗马人和凯尔特人长期冲突的开端。

这种艰苦的对抗可以在伊比利亚凯尔特人与迦太基结盟对抗罗马人的第二次布匿战争中可以看到。伊比利亚凯尔特人成功抵抗罗马人的征服直至公元前1世纪最终被帝国降伏并被并入帝国，造成许多凯尔特人在西班牙影响的残余被基本毁灭。

在将近一个半世纪后，在北面的大陆，传奇的将军尤里乌斯凯撒于公元前50年将他的军团开至高卢的荒野，发动了高卢战争。出于对在高卢扩张领土的激情和罗马人用劫掠的领土来偿还债务（老兵军团并不便宜）的倾向，高卢战争是独立高卢的结束，在此之后它被高度“罗马化”。而这些征服则是罗马最终支配大部分中欧的开端。

在凯撒领导下罗马人对不列颠展开了第一次远征，据说这是对不列颠人协助高卢人的报复。不顾激烈的抵抗，在皇帝克劳狄乌斯和之后哈德良的领导下，罗马人继续了对不列颠一个多世纪的入侵。

凯尔特文化的影响

随着凯尔特人的广泛移民和最终的被罗马社会所吸收，许多独特的凯尔特元素可以在罗马的艺术作品中被发现。在他们的高度，凯尔特人被认为也在他们的作品中采纳了许多罗马，希腊和日耳曼方面的艺术元素。随着他们被罗马人征服，许多这样被凯尔特人采纳的元素又重新被认为是罗马人的。以对动物，神祗和自然有着独特风格的演绎为特色的凯尔特文化经常被发现雕刻于罗马的陶器和雕刻之中。凯尔特人对战争艺术的发明以及风格独特的剑和铠甲的设计的发展做出贡献。据称这些设计对罗马短剑和斯巴达剑的发展产生了影响。

凯尔特的宗教

尽管我们只有很少的关于凯尔特人古代宗教的历史记录，那个时代的祭祀，即现在所知道的德鲁伊，早已成为历史学家和一般大众的兴趣和争论的主题。由希腊人和罗马人在公元前1世纪黑暗时代描述的在罗马人征服了欧洲后德鲁伊教被认为已经被压制并最终被遗忘。讽刺的是尽管罗马人拥有残酷的征服史和以取人性命为娱乐活动的放松方式，他们认为据称被德鲁伊承认的凯尔特宗教采用人牲是野蛮的。是否德鲁伊像罗马人记录中所暗示的那样频繁的使用人牲在今天依然是争论的主题。

凯尔特的语言

包含着凯尔特语言的印欧语系，一种有着几百种分支的语系，自从石器时代起就被欧洲和部分亚洲被广泛使用。尽管凯尔特语主要被发现于今天的苏格兰，爱尔兰，大不列颠和法国，在某种程度上他们被广泛地使用并被用于联系着散布在整个欧洲大陆的多元的凯尔特部落。

今天，有六种“活着的”凯尔特语言还在使用中（大部分随着时间的流逝而消亡了）。主要在威尔士和英格兰地区使用的威尔士语是剩余的语支中如今使用的最广泛的一种，据称有600,000使用者。爱尔兰语，布列塔尼语，苏格兰盖尔语，康沃尔凯尔特语和马恩岛盖尔语组成了如今可以发现的残存的凯尔特语，尽管使用这几种语支的人口不足2百万，而完全作为母语的人则更少。

现代人的祖先

今天，以推广凯特人传统和文化为目的的非政府组织“凯尔特人联盟”认定凯尔特人民族为布列塔尼，康沃尔，爱尔兰，马恩岛，苏格兰和威尔士。尽管这些地区被看作凯尔特人语言和传统的主要堡垒，在整个世界估计有数以几百万计的人以凯尔特人为祖先。6种幸存的凯尔特语言在相应的六个地区仍然被使用着。

逸闻

凯尔特绳纹艺术，被普遍认为是一种凯尔特人贡献的艺术主题，也在罗马人和以后的基督教文化中被发现。这种连续交织的并且经常无尽的打结模式至今仍被普遍认为是凯尔特文化的标志。

古代凯尔特人不太可能用“凯尔特人”这个现在被用于指代他们的术语称呼他们自己。根据历史考证希腊语"Keltoi"和拉丁语"Celtae"是现代名称的可能起源。

链甲，这种在黑暗的中世纪发展起来的许多欧式铠甲的前身被误认为是公元前3世纪凯尔特人的发明。
